# Desafío Final: Piedra, Papel o Tijera 🎮

¡Bienvenido al desafío final del Nivel 2! Esta es una aplicación **Full Stack** que permite jugar al clásico Piedra, Papel o Tijera en tiempo real contra otro jugador, utilizando una arquitectura distribuida.

## 🚀 Demo
Podés probar la aplicación funcionando acá:
👉 [https://nicososaok.github.io/desafio-ppt-final/](https://nicososaok.github.io/desafio-ppt-final/)

## 🛠️ Tecnologías utilizadas

### Frontend
- **TypeScript** & **HTML/CSS**
- **Parcel** (Bundler)
- **Custom Components** (Arquitectura basada en componentes propios)
- **State Management** (Patrón para manejar el estado global)

### Backend
- **Node.js** & **Express**
- **TypeScript**
- **Firebase Realtime Database** (Para el juego en tiempo real)
- **Firestore** (Para persistencia de usuarios y salas)
- **Render** (Hosting del servidor)

## 🕹️ Cómo jugar
1. Ingresá tu nombre.
2. Elegí entre "Nuevo Juego" o "Ingresar a una sala".
3. Compartí el ID de la sala con un amigo.
4. ¡Ambos elijan su jugada y vean quién gana!
